# Encoding & Embedding Notebook Collection

A practical collection of NLP preprocessing and feature representation notebooks using Python.

## What’s included

- `one-hot-encoding.ipynb` — One-hot encoding for word-level categorical representation.
- `bag_of_words.ipynb` — Bag of Words (BoW) for frequency-based text representation.
- `encoding-embedding.ipynb` — A broader notebook for text encoding and embedding techniques.

## Techniques covered

- One-hot encoding
- Bag-of-Words (BoW)
- TF-IDF
- Word embeddings and other vector-based text representations

## Why this project

This repository demonstrates how to convert raw text into machine-readable numeric features for NLP and machine learning workflows. It is useful for recruiters to see hands-on experience with text preprocessing and feature engineering.

## How to use

1. Create or activate your Python environment.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Open the notebooks in Jupyter or VS Code:
   ```bash
   jupyter notebook
   ```

## Notes

- The notebooks are designed for learning and demonstration purposes.
- Dense output is used in examples for easy readability in notebook format.
- For larger datasets, sparse representations should be used for efficiency.
