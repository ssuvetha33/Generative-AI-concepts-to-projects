# Generative AI Concepts to Projects

A curated collection of **Generative AI concepts** with corresponding **hands-on projects** to bridge theory and practical implementation.

## Objectives

- Understand core Generative AI concepts
- Build practical projects for each concept
- Maintain clear, reproducible implementations
- Document learnings, experiments, and outcomes

## Repository Structure

- **Concepts**: notes, theory, and references
- **Projects**: hands-on implementations
- **Research Papers**: supporting papers and reading material
- **Utilities**: helper scripts, configs, and reusable code

## Topics Covered

- Embeddings
- Text Encoding
- Vector Representations
- Prompt Engineering
- Retrieval-Augmented Generation (RAG)
- LLM Applications
- Fine-tuning concepts
- Real-world Gen AI use cases

## Project Workflow

1. Learn the concept
2. Implement a small project
3. Test with sample data
4. Document observations
5. Improve and extend

## Tech Stack

- Python
- Jupyter Notebook
- VS Code
- Git and GitHub

## Getting Started

Clone the repository:

```bash
git clone https://github.com/ssuvetha33/Generative-AI-concepts-to-projects.git
cd Generative-AI-concepts-to-projects
```

Create a virtual environment:

```powershell
python -m venv .venv
.\.venv\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```
