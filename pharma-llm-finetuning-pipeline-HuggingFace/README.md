# Pharma LLM Fine-Tuning Pipeline (Hugging Face)

This project folder packages the complete pharma-domain LLM training workflow 
End_to_End_LLM_Fine_Tuning_Non_Instruction,_Instruction,_and_Preference_Data_with_Huggingface.ipynb

## What This Pipeline Covers

1. Stage 1: Non-instruction domain adaptation (continued pretraining style)
2. Stage 2: Instruction fine-tuning with fresh LoRA adapters
3. Stage 3: Preference optimization (DPO)
4. Adapter merging and inference checks
